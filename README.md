# Snake-Game
A Snake Game in Java 13 using Swing.</br>
The Fruits are random emojis 🍇🍈🍉🍊</br>
Has a Score, Background Music, a Game Over screen and Restarts pressing Enter.</br>*Need some improvements* </br>
